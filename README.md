# CrudApiPokemon

Aplicación Android en **Java** con **PokeAPI** y **SQLite** para capturar y gestionar Pokémon.

## Características

- Explorar Pokémon desde [PokeAPI](https://pokeapi.co)
- Buscar por nombre
- Ver detalle (tipos, altura, peso, habilidades)
- Capturar en Pokédex local (SQLite)
- Editar apodo y notas
- Liberar Pokémon (eliminar)

## Requisitos

- Android Studio
- Android SDK 34
- API 24+
- Conexión a internet para explorar

## Ejecutar

1. Abrir el proyecto en Android Studio
2. Sync Gradle
3. Run en emulador o dispositivo

## Tecnologías

- Java + XML
- SQLiteOpenHelper
- HttpURLConnection (PokeAPI)
- Glide
- Material Design (estilo iOS moderno)
